# Capture

A generic inbox plugin for TiddlyWiki. Stream-based quick capture with triage to typed entities.

## Features

- **Inbox stream** — rapid keyboard-driven capture via the streams plugin
- **Triage view** — review captures, promote to typed entities
- **Configurable** — set which stream tiddler serves as inbox
- **Generic** — works with any typed model

## Requirements

- TiddlyWiki 5.3.0+
- [Streams plugin](https://saqimtiaz.github.io/streams/) by Saqimtiaz
- [Typed plugin](https://github.com/rimir-cc/tw-typed) by rimir
